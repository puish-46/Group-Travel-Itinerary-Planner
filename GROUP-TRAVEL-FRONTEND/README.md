# The Crew Canvas
### Collaborative Group Travel Planning Platform

A collaborative workspace built using the MERN stack (React, Zustand, Express, MongoDB, Tailwind CSS) that allows groups to plan trips, design day-by-day itineraries, vote on options, track and split expenses, and share photos.

## Features
- **Shared Itinerary Builder**: Real-time collaborative schedule planning.
- **Poll System**: Group voting on lodging, dining, and activities.
- **Expense Splitter**: Log transactions and calculate headshares.
- **Shared Photo Gallery**: Shared memory board backed by Cloudinary.
- **Member Management**: Trip organizers can add members to their canvas.
